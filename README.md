# npm-package-ts-builder-boilerplate

**Best way to start to build npm package!**

The boilerplate provides two different options to choose from.
1: **main branch** - The whole setup with all needed to start building npm packages: typescript, SCSS files, HTML files.
2: **minimal-setup branch** - Setup only for typescript. If you don't use html, css files better to choose this one.
